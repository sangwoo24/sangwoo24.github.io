    ⛄️ Parking System


  • - 4학년 1학기 고급 임베디드 시스템의 기말 Term Project(2인 1조).

    - Achro.imx6q 보드를 이용해 Ubuntu Linux 환경에서 자율 프로젝트 진행.

    - Achro.imx6q가 주차 정산기가 되고, Server인 Raspberry가 차량의 번호판을 인식하여 정보를 Client인 Achro.imx와 Android에 넘겨준 후 출차/주차 등의 정산 프로그램 진행

    - 차량의 번호판 인식은 Raspberry pi Opencv의 Tesseract를 이용.

    - Android 프로그래밍은 Android Studio를 이용해 제작. - Achro.imx6q를 이용하여 Kernel수준의 Device Driver를 제작, 내부의 8가지 모듈과 외부 센서[초음파]를 이용.

    - Achro.imx의 터치스크린 LCD와 Android의 View를 이용하여 가상의 주차장 시뮬레이터 생성.
