    🚗 Parking System

  • 개발 환경

    - C Language,   Python,   Java,   OpenCV

    사용 통신 규격

    - Socket Communication,   WIFI,   Serial Communication

    내용 및 기능

    - 4학년 1학기 고급 임베디드 시스템의 기말 Term Project(2인 1조).
    - Achro.imx6q 보드를 이용해 Ubuntu Linux 환경에서 자율 프로젝트 진행.
    - Achro.imx6q를 이용하여 Kernel수준의 Device Driver를 제작, 내부의 8가지 모듈과 외부 센서[초음파]를 이용.
    - Achro.imx의 터치스크린 LCD와 Android의 View를 이용하여 가상의 주차장 시뮬레이터 생성.

    차량 인식 : Server인 Raspberry pi는 Achro.imx6q가 초음파 센서로 차량을 인식할 시 Opencv Tesseract로 차량의 번호판을 추출하고, Client인 Android와 Achro.imx6q에게 차량 정보를 전송.

    Android : Android는 정보를 받은 뒤 DB에 저장을 하여 차량조회, 요금조회, 주차 정산의 기능을 수행.

    Achro.Imx6q : Achro.imx6q는 차량을 초음파 센서로 감지한 뒤, 정보를 받고 차량정산, 요금조회, 주차정산의 기능을 수행. 프로젝트 세부내용




    🚗 Graduation Project [HandTalker]

  • 개발 환경

    - C,   Python,   Java,   Arduino

    사용 통신 규격

    - Socket Communication,   Bluetooth,   Serial Communication

    내용 및 기능

    - 전자공학부 졸업 작품으로 장갑을 착용하고 수화를 하면 수화의 음성을 추출해주는 작품을 개발.

    - Bluetooth 통신이 가능한 Jarduino와 Android 사이에서 Data교환이 이루어짐.

    - 오른손은[Ebimu : 6, Flex : 6, Capacitor : 2]의 14개의 센서가 들어가고, 왼손에 경우에는[Ebimu : 6, Flex : 5]의 11개의 센서가 들어간다.

    - Android는 Arduino로 부터 Data를 받고, 가속도 기반 에너지를 추출하여 수화의 동작 구간을 판별.

    - 에너지 전위가 임계치를 넘으면 동작으로 인식하고, 지화는 저장된 Data와 손의 끝점을 비교해 추출한다.

    - 수화에 경우에는, 손목, 접촉센서, 끝점에 따라 reject를 한 후 DTW(Dynamic Time Wraping) 알고리즘을 이용해 구간에 대한 Data를 비교한 후 가장 유사한 패턴의 수화를 출력한다.

    - 오른손의 에너지만 올라간 경우에는 지화 및 한손수화로 인식하고, 왼손의 에너지가 먼저 올라가거나 양손의 에너지가 임계치를 넘으면 수화 동작으로 인식한다.
